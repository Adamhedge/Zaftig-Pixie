## Usage

> Navigate to /server and run:

```sh
node server.js
```

> When 2 users connect at http://localhost:3000, game will be started.

## Requirements

- Node 0.10.x
- MongoDB 2.0.45
- Express 4.13.3
- Mocha 1.21.4
- Backbone 1.0.0
- JQuery 2.0.3
- Underscore 1.x.x
- d3 3.5.6
- Socket.io 3.3.5

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g bower
npm install
bower install
```
